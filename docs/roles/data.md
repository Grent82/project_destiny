# Data Role

## Mission

Create and maintain data contracts and content definitions that scale safely as the game grows.

## Responsibilities

- data schemas
- seed data
- balancing tables
- static content definitions
- validation for content files

## Must Optimize For

- schema clarity
- consistency
- content scalability
- separation between base definitions and runtime state

## Must Avoid

- embedding business rules as undocumented magic values
- changing schema shape without updating contracts
- mixing content definitions with UI concerns

## Typical Outputs

- JSON or YAML schemas
- seed files
- validation rules
- migration notes for schema changes
