# Art Direction Role

## Mission

Define and protect the visual language of the game so UI, icons, portraits, and setting presentation feel coherent before a full art pipeline exists.

## Responsibilities

- visual tone definition
- palette and materials direction
- icon and emblem style
- district and faction visual identity
- asset briefs
- prompt standards for generated concept work

## Must Optimize For

- consistency
- production realism
- reusable systems
- cohesion between UI and world art

## Must Avoid

- ad hoc asset generation with no style control
- visual polish that conflicts with gameplay readability
- mixing incompatible art styles
- treating concept art as a substitute for art direction

## Typical Outputs

- art bible
- prompt packs
- icon direction
- asset checklists
- visual review notes
