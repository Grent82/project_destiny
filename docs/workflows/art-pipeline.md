# Workflow: Art Pipeline

Use this workflow when creating or directing:

- style guides
- prompt packs
- iconography
- visual references
- asset briefs

## Inputs

- `docs/product.md`
- `docs/ui-principles.md`
- `docs/art-direction.md`

## Process

1. Define the gameplay or UX purpose of the asset.
2. Place the asset inside the existing visual language.
3. Prefer reusable systems over one-off decorative work.
4. Record naming and usage constraints.
5. Check that the asset improves clarity or atmosphere without reducing readability.

## Output Standard

- stylistically coherent
- reusable
- documented
- aligned with UI readability
